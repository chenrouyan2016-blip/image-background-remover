# Image Background Remover — MVP 需求文档

**版本：** v0.1  
**日期：** 2026-03-29  
**目标：** 快速上线可用产品，验证用户需求

---

## 一、产品概述

### 产品定位
一款在线 AI 图片背景去除工具，用户上传图片即可一键抠图，无需注册，免费使用（限制分辨率），支持高清下载付费升级。

### 目标用户
- 电商卖家（商品图处理）
- 设计师（快速抠图）
- 普通用户（证件照、头像处理）

### 核心价值
- **快**：秒级出结果
- **准**：Remove.bg API 保障质量
- **简**：无需注册，拖拽即用

---

## 二、功能范围（MVP）

### ✅ 包含功能

| 功能 | 优先级 | 说明 |
|------|--------|------|
| 图片上传 | P0 | 支持拖拽 / 点击上传，格式 JPG/PNG/WEBP，最大 10MB |
| 背景去除 | P0 | 调用 Remove.bg API 处理，全程内存，不落盘 |
| 结果预览 | P0 | 原图 / 结果图左右对比展示 |
| 图片下载 | P0 | 下载 PNG（透明背景） |
| 免费限制 | P0 | 免费版输出预览分辨率（约 600px），高清需付费 |
| 多语言 | P1 | 中英文切换（上线时优先英文） |
| 批量处理 | ❌ | MVP 不做，后期迭代 |
| 背景替换 | ❌ | MVP 不做，后期迭代 |
| 用户账户 | ❌ | MVP 不做，后期迭代 |

### 页面清单

```
/ 首页          — 上传入口 + 功能介绍 + 示例展示
/result         — 结果预览 + 下载（处理完成后跳转）
/pricing        — 定价页（占位，后期完善）
```

---

## 三、用户流程

```
1. 用户访问首页
2. 拖拽或点击上传图片（JPG/PNG/WEBP，≤10MB）
3. 系统提示"处理中..."（loading 状态）
4. 调用后端 API → 转发 Remove.bg → 返回结果
5. 跳转结果页，左右对比展示原图与抠图结果
6. 用户点击"下载 PNG"
   ├─ 免费：下载预览尺寸（600px）
   └─ 付费：下载原始高清（后期）
```

## 四、技术方案

### 技术栈
| 层级 | 技术 |
|------|------|
| 前端框架 | Next.js 14（App Router） |
| 样式 | Tailwind CSS + Shadcn/ui |
| 后端 | Next.js API Routes（Serverless） |
| 图片处理 | 内存 Buffer，不落盘 |
| AI 能力 | Remove.bg API |
| 部署 | Vercel |
| 域名 | 自定义域名（后配） |

### API 接口

**POST `/api/remove-bg`**

| 参数 | 类型 | 说明 |
|------|------|------|
| image | File | 原始图片文件 |

返回：PNG 二进制流（透明背景）

### 环境变量
```
REMOVE_BG_API_KEY=xxx
```

## 五、非功能需求

| 项目 | 要求 |
|------|------|
| 处理时长 | ≤ 5 秒（Remove.bg 正常响应） |
| 文件大小限制 | ≤ 10MB |
| 支持格式 | JPG、PNG、WEBP |
| 移动端适配 | 响应式，手机可用 |
| 隐私 | 图片不存储，处理后即丢弃 |
| SEO | 首页关键词：image background remover, remove background free |
